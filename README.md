# 🐕 Ddaraogae(따라오개)

```
"Ddaraogae"는 반려견 산책을 기록하고, 도와주며 견주에게는 편의성과 재미를 제공하는 앱 입니다.
```

</br>

## 1. 프로젝트 소개
<aside>
  
```
💡 반려견과 견주 모두에게 즐거운 산책
사랑하는 반려견과 얼마나 산책 했을까요? 산책 그래프와 경로를 한 눈에 확인해보세요!
```

**따라오개**는 반려견과의 산책을 기록하는 앱입니다. 산책 이력과 경로를 시각적으로 확인하고 관리할 수 있습니다.

산책 이력을 기반으로 획득할 수 있는 산책 도장을 모아 성취감을 느낄 수 있습니다.
